Um sistema completo de **Cardápio Digital Responsivo** projetado para restaurantes, lanchonetes e hamburguerias. Este projeto permite que o estabelecimento receba pedidos de forma automatizada e organizada diretamente no WhatsApp, eliminando a dependência e as altas taxas de plataformas de delivery (como o iFood).

## ✨ Funcionalidades Principais (Features)

*   📱 **Mobile-First UI/UX:** Interface premium desenvolvida com foco total na experiência mobile (onde 99% dos pedidos acontecem).
*   🔍 **Filtros Dinâmicos:** Navegação instantânea entre categorias (Todos, Burgers, Porções, Bebidas) sem recarregar a página.
*   🛒 **Carrinho de Compras Inteligente:**
    *   Soma automática de itens e valores.
    *   Acréscimo e decréscimo de quantidades (+ / -) direto no modal.
    *   Remoção automática do item caso a quantidade chegue a zero.
    *   O botão de "Meu Pedido" flutuante só aparece quando há itens selecionados.
*   💬 **WhatsApp Checkout:** Validação de endereço de entrega e envio do pedido formatado, somado e detalhado direto para o WhatsApp do restaurante.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as bases sólidas do desenvolvimento Web, sem dependência de frameworks pesados, garantindo carregamento instantâneo e alta performance:

*   **HTML5:** Estrutura semântica e acessível.
*   **CSS3:** Variáveis nativas (`:root`), Flexbox, CSS Grid, animações (`@keyframes`) e design Glassmorphism.
*   **JavaScript (Vanilla):** Manipulação avançada do DOM, Array Methods (`map`, `filter`, `find`, `reduce`) e lógica de carrinho e estado.
